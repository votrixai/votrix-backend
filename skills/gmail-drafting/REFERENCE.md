# Email Drafting Reference

## Greeting templates

| Relationship       | Greeting                          |
|--------------------|-----------------------------------|
| Formal / unknown   | Dear [Name],                      |
| Business / known   | Hi [Name],                        |
| Colleague / casual | Hey [Name], / [Name] —            |
| No name known      | Hello, / Hi there,                |

---

## Sign-off templates

| Context            | Sign-off                          |
|--------------------|-----------------------------------|
| Formal             | Best regards, / Sincerely,        |
| Standard business  | Best, / Thanks,                   |
| Casual             | Cheers, / Talk soon,              |
| Action expected    | Looking forward to hearing from you, |

---

## Common subject line patterns

```
[Action needed] <topic> — <deadline>
Re: <original subject>          ← for replies (keep original)
<Topic>: <specific detail>
Quick question: <topic>
<Name>, <brief ask>
```

---

## Body structure

```
[One-line context or reason for writing]

[Core message — what you need or are sharing]

[Call to action or next step]

[Sign-off]
[Name]
```

---

## Phrase bank

**Opening (skip filler — go straight to the point):**
- "I'm writing to …"
- "Following up on …"
- "Sharing an update on …"

**Requests:**
- "Could you please … by [date]?"
- "Would it be possible to …?"
- "Let me know if you need anything else."

**Closing CTAs:**
- "Happy to jump on a call if easier."
- "Please let me know your thoughts."
- "Feel free to reach out with any questions."
