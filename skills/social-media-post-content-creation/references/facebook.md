# Facebook 内容规格

---

## 内容类型总览

| 类型 | 适合场景 | 触达模式 |
|---|---|---|
| Feed Post | 活动公告、产品推广、品牌故事、链接分享 | 粉丝 feed，算法分发 |
| Reels | 短视频内容，触达新用户 | 算法推流，可触达非粉丝 |
| Story | 限时活动、互动、日常更新 | 粉丝 story 栏，24h 消失 |

---

## Feed Post（动态帖子）

**适用：** 产品推广、活动通知、行业知识、客户案例、链接分享

**媒体规格：**
- 图片：1:1（1200×1200）/ 16:9（1200×630），JPG/PNG，最大 10MB
- 视频：最长 240 分钟，MP4，建议 16:9 或 1:1
- 最多 10 张图片（相册形式）

**文案结构：**
```
[Hook — 第一句，40 字以内效果最好]

[正文 — 叙事感，可适当长，有料有情感]

[CTA — 明确下一步]

[链接 — 直接贴，Facebook 会自动生成预览卡]

[Hashtag — 3–5 个，效果不如 IG 明显，适量即可]
```

**写作要点：**
- 40–80 字互动率最高；长文需要强 hook，否则被折叠
- 贴链接会自动生成预览卡，正文不需要重复描述链接内容
- 原生上传视频比贴 YouTube 链接触达更广
- Facebook 用户偏好叙事感和情感共鸣，不要太硬销
- Hashtag 在 Facebook 效果弱，不必堆数量

**配图 aspect_ratio：** `"1:1"` 或 `"16:9"`

---

## Reels（短视频）

**适用：** 品牌曝光、触达新用户、产品演示、轻松娱乐内容

**视频规格：**
- 比例：9:16（1080×1920），竖屏
- 时长：3s–90s（30–60s 效果最好）
- 格式：MP4

**文案结构：**
```
[Caption — 简短，50 字以内]
Reels 靠视频内容传达，caption 辅助

[Hashtag — 3–5 个]
```

**写作要点：**
- Facebook Reels 与 Instagram Reels 可用同一视频素材跨发布
- caption 可比 IG 稍长，但仍以简洁为主
- 链接在 Facebook Reels 里可点击（与 IG 不同）

**配图 aspect_ratio：** `"9:16"`（封面图）

---

## Story（限时动态）

**适用：** 限时优惠、日常更新、幕后花絮、简单公告

**规格：**
- 比例：9:16（1080×1920）
- 上下各留 14% safe zone
- 持续时间：24 小时消失
- 图片显示 5s，视频最长 20s

**无持久 Caption**，信息靠图片 / 视频本身传达。

**agent 需要生成：**
```
视觉内容描述：[画面主体]
文字叠加（如有）：[简短标语，最多 2 行]
互动贴纸（如有）：Poll / Question / 无
```

**写作要点：**
- Facebook Story 互动率低于 Instagram，不作为核心内容渠道
- 适合配合 Feed Post 做补充曝光（同日内容同主题）
- 关键文字 / logo 放中间 72%

**配图 aspect_ratio：** `"9:16"`

---

## 最佳发布时间

| 类型 | 推荐时间 |
|---|---|
| Feed Post | 周三、四，上午 10–11 点 |
| Reels | 周一–周五，下午 1–4 点 |
| Story | 早 8–9 点 或 晚 7–9 点 |

[3–5 个]
